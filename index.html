Featured

Personal Trainer

A frontend react application for a customers and trainings database.

The app was built with axios for communication with the backend and mainly uses the material-ui react component library to build CRUD (Create-Read-Update-Delete) functionalites for customers and trainings. Client side routing was built with react-router-dom and testing was done using jest. Deployment is done through Heroku.

Other libraries used to build the User Interface (UI) were recharts to show the statistics, react-big-calendar and date-fns, ag-grid-react to show the customer and training lists.

Blog List
